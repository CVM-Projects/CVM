# CVM
The first Virual Machine for CMS language

[![Build Status](https://api.travis-ci.org/CVM-Projects/CVM.svg?branch=master)](https://travis-ci.org/CVM-Projects/CVM)

## Build

```
mkdir build
cd build
cmake ..
make
```

## Run

```
./cvm ../test.cms
```
